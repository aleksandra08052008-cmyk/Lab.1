## Тема: 
Масиви в мові програмування Java

## Мета: 
Ознайомлення з масивами та використання основних
методів їх обробки в мові програмування Java. Здобуття
навичок у використанні масивів в мові програмування Java.

## Варіант завдання
1) С5 = 0 => C = a · B

3) С11 = 5 =>
 Обчислити суму найбільших
елементів в рядках матриці з

## Опис
1. Створити символьну матрицю B.
2. Помножити всі елементи матриці на число a.
3. Результат записати у матрицю C.
4. Знайти:
   - максимальний елемент у парних рядках;
   - мінімальний елемент у непарних рядках.
5. Обчислити суму знайдених елементів.
6. Написати модульні тести для перевірки роботи програми

У програмі використовується двовимірний масив типу char.

Кожен елемент матриці B множиться на число a, після чого результат записується у матрицю C.

## Опис тестів
testMatrixMultiplication()

Перевіряє правильність множення елементів матриці на число.
assertEquals((char) ('A' * 2), C[0][0]);

testSumLogic()

Перевіряє коректність логіки знаходження суми максимальних та мінімальних елементів.
assertTrue(sum > 0);

```java
C[i][j] = (char) (B[i][j] * a);

__Приклад програми__
Матриця B:
A B
C D

Множник a = 2

Матриця C після множення:
130 132
134 136

Обчислення суми:
Рядок 0 → max = 132
Рядок 1 → min = 134

Сума = 266
java'''
